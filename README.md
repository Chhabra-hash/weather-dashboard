# 🌦️ Weather Dashboard

A modern and responsive Weather Dashboard built using **HTML5, CSS3, and Vanilla JavaScript**. This project features a secure login system that authenticates users through a local `users.json` file and provides real-time weather updates with a 3-day forecast using the **WeatherAPI**.

Designed as a mini web development project, it demonstrates core front-end concepts such as API integration, asynchronous JavaScript, responsive UI design, DOM manipulation, and Git version control.

---

## ✨ Features

- 🔐 Secure Login Authentication
- 📂 User validation using `users.json`
- 🌍 Search weather by city name
- 🌡️ Live temperature display
- ☁️ Current weather condition with icon
- 💧 Humidity information
- 💨 Wind speed display
- 📅 3-Day weather forecast
- 🎨 Dynamic UI based on weather conditions
- 🚪 Logout functionality
- ❌ Error handling for invalid city names
- 📱 Fully responsive design for desktop and mobile

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Chhabra-hash/weather-dashboard.git
   ```

2. Open the project in Visual Studio Code.

3. Run the project using the **Live Server** extension.

4. Log in using the demo credentials and search for any city to view live weather information.
